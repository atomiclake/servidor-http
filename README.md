# Servidor HTTP
Implementação de um servidor HTTP usando a biblioteca de Sockets do C#.

Este software foi criado para fins de estudo e não deve ser usado em produção.

## Funcionalidades
| Descrição | Status |
|-----------|--------|
| Método GET | Pronto ✔ |
| Resposta 404 | Pronto ✔ |
| Exceções tratadas com erro 500 | Em andamento ⏰ |
